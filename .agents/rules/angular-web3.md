## trigger: always_on

Yêu cầu mô tả, các bước thực hiện, kế hoạch, scratchpad, implementation plan, task đều phải ghi tiếng việt trong Chatbox Antigravity

Những gì tôi yêu cầu tại thư mục dự án, bạn hãy ghi nhớ những gì tôi yêu cầu, những gì bạn đã thực hiện, quá trình và lưu vào file AGENTS.md tại thư mục gốc .agent\AGENTS.md
